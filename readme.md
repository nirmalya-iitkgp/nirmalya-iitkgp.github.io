Nirmalya Panigrahi | Interactive Digital Portfolio
A high-performance, responsive personal portfolio and app showcase built with a modern "Vanilla" stack (HTML5, CSS3, JavaScript). This project features a custom theme engine, interactive particle backgrounds, and a centralized project navigation system designed for speed and aesthetic precision.

✨ Key Features
Dynamic Theme Engine: Swap between four distinct visual modes (Classic, Serene, Aurora, Cosmos) that update primary colors, typography gradients, and particle physics in real-time.

Centralized App Hub: A custom-built modal system for navigating SaaS platforms, AI games, and professional tools.

Interactive Backgrounds: Integrated with tsParticles for a responsive, interactive node-link network that reacts to theme shifts.

Typing Animation: Engages visitors with a professional introduction powered by Typed.js.

Infinite Scrolling Marquee: A seamless loop of professional partners and educational background in the footer.

Modern Iconography: Transitioned from SVG-heavy libraries to Font Awesome 6 for improved load times and reliability.

🛠️ Built With
HTML5 & CSS3: Core architecture and custom keyframe animations.

Tailwind CSS (CDN): Rapid utility-first styling and responsive grid layouts.

JavaScript (ES6+): Custom state management for themes and UI interactions.

Font Awesome: Comprehensive icon set for professional and social links.

tsParticles: High-performance canvas-based particle animations.

Typed.js: Dynamic string animation for hero descriptions.

🎨 Visual Themes
The UI adapts to four distinct palettes:

Classic: High-contrast dark mode (McKinsey-inspired professionalism).

Serene: Slate and sky-blue tones for a tech-focused feel.

Aurora: Deep emerald and mint accents inspired by nature.

Cosmos: Violet and purple highlights for a vibrant, creative aesthetic.

🔧 Installation & Setup
Since this project uses a vanilla stack with CDN-hosted dependencies, no installation or build steps (like npm install) are required.

Clone the repository:

Bash
git clone https://github.com/nirmalya-iitkgp/portfolio-v1.git
Open in Browser:
Simply open index.html in your browser.

Local Development:
It is recommended to use the Live Server extension in VS Code to ensure all SVG assets and scripts load without CORS issues.

📝 Configuration
To update your own links and projects, modify the constants at the top of script.js:

SOCIAL_MEDIA_LINKS: Update URLs and Font Awesome icons.

APPS: Add new SaaS or project links with custom colors and icons.

PARTNER_LOGOS: Update with your own university or company SVG logos.
