# Nirmalya Panigrahi | Interactive Digital Portfolio

A high‑performance, responsive personal portfolio and app showcase built with a modern **Vanilla stack** (HTML5, CSS3, JavaScript).  
This project features a custom theme engine, interactive particle backgrounds, and a centralised project navigation system designed for speed and aesthetic precision.

## ✨ Key Features

### 🎨 Dynamic Theme Engine
Swap between four distinct visual modes (**Classic, Serene, Aurora, Cosmos**) that update:
- Primary colours  
- Typography gradients  
- Particle physics  
in real time.

### 🧭 Centralised App Hub
A custom‑built modal system for navigating:
- SaaS platforms  
- AI games  
- Professional tools  

### 🌌 Interactive Backgrounds
Integrated with **tsParticles** for a responsive, interactive node‑link network that reacts to theme shifts.

### ⌨️ Typing Animation
Engages visitors with a professional introduction powered by **Typed.js**.

### 🔁 Infinite Scrolling Marquee
A seamless loop of professional partners and educational background in the footer.

### 🎯 Modern Iconography
Transitioned from SVG‑heavy libraries to **Font Awesome 6** for improved load times and reliability.

## 🛠️ Built With

- **HTML5 & CSS3**: Core architecture and custom keyframe animations  
- **Tailwind CSS (CDN)**: Utility‑first styling and responsive grid layouts  
- **JavaScript (ES6+)**: Custom state management for themes and UI interactions  
- **Font Awesome**: Icon set for professional and social links  
- **tsParticles**: High‑performance canvas‑based particle animations  
- **Typed.js**: Dynamic string animation for hero descriptions
