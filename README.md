# 🌌 Nirmalya Panigrahi | Personal Portfolio

Welcome to my interactive digital space. This project is a high-performance, visually immersive portfolio built to showcase my journey, projects, and professional experience.

## 🚀 Key Features

- **🎭 Dynamic Atmosphere:** Integrated `tsParticles` with custom-tuned physics for a living background that responds to your touch and movement.
- **🌈 Immersive Color Modes:** A unique theme-switching engine allowing users to toggle between 4 distinct visual identities:
  - **Classic Dark:** The foundational minimalist aesthetic.
  - **Serene Blue:** Calm, deep ocean vibes for focused reading.
  - **Aurora Green:** Electric, high-contrast energy.
  - **Cosmos Purple:** Mystical, expansive celestial tones.
- **✨ Fluid Typography:** Smooth text animations using `Typed.js` that capture attention and guide the narrative.
- **📱 Responsive by Design:** A seamless experience across all viewports, from ultra-wide desktops to mobile devices.
- **⚡ Performance-First:** Built with React 18, Vite, and Tailwind CSS for near-instant load times and buttery-smooth animations.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Animations:** [Motion](https://motion.dev/) (f.k.a. Framer Motion)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Visuals:** [tsParticles](https://particles.js.org/)
- **Typography:** [Typed.js](https://mattboldt.github.io/typed.js/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🎨 Design Philosophy

This portfolio follows a **minimalist-brutalist** hybrid approach. It avoids unnecessary clutter while using bold typography and high-fidelity interactive elements to provide a "signature" feel.

- **Typography:** Inter & custom Serif pairings for a balanced technical-editorial look.
- **Interaction:** Every click triggers a visual response, making the act of exploring the site feel like a game.
